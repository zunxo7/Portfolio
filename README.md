## ğŸ¨ Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --accent-primary: #ff6b35;
    --bg-primary: #fafafa;
    --text-primary: #0a0a0a;
    /* ... more variables */
}
```

### Typography
Fonts are loaded from Google Fonts. Modify in `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@...&family=Poppins:wght@..." rel="stylesheet">
```

### Content
All content is in `index.html`. Simply edit the HTML to update text, links, and sections.
